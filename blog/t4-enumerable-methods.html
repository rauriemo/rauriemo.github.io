Enumerable Map Method

Feb. 9th, 2015

The Enumerable class functions much like an each method that iterates through every element of a given collection and has several methods to manipulate this data.

In order to use the interation powers of the Enumerable class you need to include a link to it inside your class formation. You do this simply by writing "include Enumerable" in the class and by defining an each method within it that yields all the elements of the desired collection to tbe iterated on.

One powerful tool that Enumerable provides is the map method. It takes a collection and a criteria and returns a copy of that collection where each element was transformed based on the criteria. The syntax for this is: collection.map{|element| element.to_s} In this example we return the elements of a collection transformed to strings. The map method is non destructive and were you to map an array you would not be altering the original content of the array.

This is very handy when you need to transform two collections into the same object class for comparison purposes allowing you to not alter the original data at all. For example if you wanted to find if the number 1 is a value within a hash you can map the hash transforming all the values into integers to see if one of the values is, even if it was entered as a string.
