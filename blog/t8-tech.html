Recursion

Mar. 8, 2015

Recursion refers to a method of solving a problem where the larger solution is composed of of several instances of the same problem but smaller. Most pogramming languages allow for recursion by allowing methods or functions to call themselves. Languages that only rely on recursion and don't employ loops or other iterative structures are just as powerful as imperative languages such as Ruby and Javascript.

One classic example of a function made simpler through recursion is in the factorial equation. Instead of having to go through every number in the equation we simply have to use the formula n(n-1) when n>0 and 1 when n=0. We can keep calling the function and it will keep multiplying by the next lowest number untill it reaches 0.

In imperative programming languages iteration is preferred for simple recursion as it avoids buildup of function calls and allows for better call stack management. In functional languages recursion is the prefered method.
