Arrays and Hashes

Feb. 2, 2015

Arrays and Hashes are are both ways of recording lists of information that are stringed together. An array is indexed by ordered integers beginning at 0. A Hash on the other hand is a collection of key-value pairs so the key is also arbitrary vs being fixed to an ordered integer sequence like in an array. Keys in a hash can be of any object class.

One practical use of hashes that make them better than arrays is when we want to index a database with actual User Names as the key. It often makes sense to have a persons name or social security/id number be their key.
